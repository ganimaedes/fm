# fm

Fm is a minimal file manager/viewer for linux

To compile with gcc:

cd src/ && make && sudo make install

To execute:

fm "$HOME"

Keys

| Keys          | Associated Action |
| ------------- | :---------------: |
| UP arrow or k | select item above |
| DN arrow or j | select item below |
| \<DEL> or dd  | delete file       |
| c or yy       | copy file         |
| p             | paste file        |
| \<Enter>      | view folder       |
